# Weird-Programs
A collection of experimental, testing, and random programs and scripts.

All code and programs in this repository are intended for educational and exploratory purposes only.

Some projects simulate common malware behaviors or include intentionally exaggerated or misleading descriptions; however, they do not contain any real malicious code.

The misuse of these tools is neither encouraged nor supported. This repository is meant solely for learning, controlled testing, and technical curiosity.

Any use of the content provided here is at your own risk.
